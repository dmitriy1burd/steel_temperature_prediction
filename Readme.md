# Оптимизация производственных расходов металлургического комбината

Металлургическому комбинату необходимо разработать модель МО, предсказывающую конечную температуру сплава на основе данных о технологическом процессе, чтобы в дальнейшем оптимизировать потребление электроэнергии

 
 Основные этапы работы:
1. Подключение к базе данных и изучение данных:
  - Установка соединения с SQLite базой данных;
  - Проверка списка доступных таблиц;
  - Выполнение анализ пропусков в каждом столбце.
2. Предобработка данных.
3. Исследовательский анализ данных:
    - Проведение статистического анализ;
    - Генерация новых признаков;
    - Выявление и обработка аномальных значений.
4. Подготовка данных для модели:
    - Создание агрегированного датафрейма на основе исходных таблиц;
    - Разделение на тренировочную и тестовую выборки (75:25)
5. Обучение моделей:
    - Сравнение нескольких моделей:  DecisionTreeRegressor, LGBMRegressor;
    - Разработка полносвязной нейросети;
    - Оптимизация гиперпараметров с использованием Optuna.
6. Тестирование лучшей модели:
    - Проверка работы лучшей модели на тестовой выборке;
    - Исследование важности признаков с использованием shap.


Результаты обучения моделей:
| Модель | MAE | MSE | R² |
|--------|-----|-----|-----|
| XGBRegressor | 6.253 | 69.780 | 0.453 |
| RandomForestRegressor | 6.799 | 89.877 | 0.301 |
| DecisionTreeRegressor | 7.315 | 102.873 | 0.197 |
| FeedForwardNN | 8.420 | 134.638 | 0.007 |
